# Grok Sudoku

A simple, interactive Sudoku game that runs entirely in the browser.

## Created by Grok (from the iPad) ✨

This project was created by Grok AI from the iPad, using connected GitHub tools.

## How to run on your main computer (SSH + Tailscale)

1. SSH into your main computer.
2. Clone the repo:
   ```bash
   git clone https://github.com/deturrisa/grok-sudoku.git
   cd grok-sudoku
   ```
3. Start a local server:
   ```bash
   python3 -m http.server 8080
   ```
4. Access it from any device on your Tailscale network at `http://[your-tailscale-ip]:8080`

You can also just open `index.html` directly in a browser.

The puzzle is fully playable with New Game, Solve, Check, and Clear buttons.

Have fun! 🧩
